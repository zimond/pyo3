[−][src]Struct pyo3::prelude::PyObject

#[repr(transparent)]
pub struct PyObject(_);

A python object

The python object's lifetime is managed by python's garbage collector.

Technically, it is a safe wrapper around NonNull<ffi::PyObject>.
